自动对账系统 - 本地一键启动

使用方式
1. 双击「一键启动.bat」
2. 首次启动会自动创建虚拟环境并安装依赖
3. 浏览器会自动打开: http://127.0.0.1:5000
4. 关闭启动窗口即可停止服务

可分享给他人
1. 把整个项目文件夹打包为 zip 发给对方
2. 对方解压后，双击「一键启动.bat」即可使用

运行要求
- Windows 10/11
- Python 3.10+（需已安装）

常见问题
1) 提示未检测到 Python
   - 请先安装 Python，并勾选 Add Python to PATH

2) 依赖安装慢或失败
   - 网络原因，重试启动脚本即可
